# Workflow 05: 테스트 및 검증

## 목표
모든 기능이 정상 동작하는지 검증한다.

## 테스트 항목

### 1. 인증 테스트
- 본인 계정 로그인/로그아웃
- 와이프 계정 로그인/로그아웃
- 다른 사용자 데이터 접근 불가 확인 (RLS)

### 2. 촬영 + AI 분석
- 카메라 촬영 → Gemini 분석 → 결과 표시
- 갤러리 선택 → Gemini 분석 → 결과 표시
- 실제 음식 3~5장 분석 정확도 확인

### 3. 수정 + 저장
- AI 결과에서 칼로리 수동 수정 → 저장
- DB에 수정된 값 반영 확인

### 4. 대시보드
- 여러 끼 입력 후 칼로리 합계 정확성
- 원형 차트 목표 대비 비율 확인

### 5. 히스토리
- 날짜별 필터링 정상 동작
- 과거 기록 조회 확인
- 수정/삭제 동작

### 6. 리포트
- 7일치 입력 후 주간 차트 정확성
- 월간 집계 정확성

### 7. 건강 가이드
- AI 조언이 프로필(목표) 기반인지 확인
- 최근 7일 식단 반영 확인

### 8. 오프라인
- 비행기 모드에서 사진 촬영 + 기록
- 네트워크 복구 후 자동 동기화

### 9. 2대 동시
- 본인 + 와이프 폰에서 동시 사용
- 데이터 분리 확인

## 에러 발생 시
- 자기 개선 루프 실행 (원인분석 → 수정 → 재테스트 → 문서 업데이트)
